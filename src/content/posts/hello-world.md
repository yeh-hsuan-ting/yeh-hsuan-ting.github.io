## How drafts work

Set `draft: true` in the frontmatter and a post is excluded from the build and
the feed. It still renders while you run `npm run dev`, so you can preview it
locally before publishing.

> Add a post locally by dropping a Markdown file in `src/content/posts/`, or
> from the web at `/admin`.
